# CS Decibel Meter

A Flutter application for measuring ambient sound levels (decibels) in real-time.

## Features

- Real-time decibel measurement
- Statistics display (min, max, average, percentiles)
- Multi-language support (English, Chinese, Spanish, Hindi, Arabic, Portuguese, Bengali, Russian, Japanese, German)
- Clean and modern UI
